# Mergeware Meteor Todo App

A Todo Management Application built using **Meteor.js**, **Blaze**, and **MongoDB** as part of the Mergeware technical assessment.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* User Authentication (Register/Login)
* Task Categories

  * Work
  * Personal
  * Urgent
  * Other
* Drag-and-Drop Task Reordering
* Task Filtering
* Responsive User Interface

## Tech Stack

* Meteor.js
* Blaze
* MongoDB
* JavaScript
* HTML
* CSS

## Installation

```bash
git clone https://github.com/Srigowrisahithi/mergeware-meteor-todo.git
cd mergeware-meteor-todo
meteor npm install
meteor
```

## Running the Application

```bash
meteor
```

The application will be available at:

http://localhost:3000

## Project Structure

* Client Side (Blaze Templates)
* Server Side (Meteor)
* MongoDB Collections
* Publications and Subscriptions
* User Authentication
* Task Management Features
